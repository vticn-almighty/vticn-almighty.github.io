# STL容器总结

STL是Standard Template Library的简称，中文名标准模板库，惠普实验室开发的一系列软件的统称。 在C++标准中，STL被组织为下面的13个头文件： <algorithm>、<deque>、<functional>、<iterator>、<vector>、<list>、<map>、<memory.h>、<numeric>、<queue>、<set>、<stack>、<utility>

经典的数据结构数量有限，但是我们常常重复着一些为了实现向量、链表等结构而编写的代码，这些代码都十分相似，只是为了适应不同数据的变化而在细节上有所出入。STL容器就为我们提供了这样的方便，它允许我们重复利用已有的实现构造自己的特定类型下的数据结构，通过设置一些模板类，STL容器对最常用的数据结构提供了支持，这些模板的参数允许我们指定容器中元素的数据类型，可以将我们许多重复而乏味的工作简化。STL容器主要分为序列式容器和关联式容器两类，前者的元素位置取决于插入顺序，而后者按照红黑树的数据结构进行插入因此元素位置与插入顺序无关。

本文参考的重要技术文档

主要STL容器介绍

容器类名 特性 何时使用
vector 在内存中占有一块连续的空间，存储一个元素序列。可以看作一个可自动扩充的动态数组，而且提供越界检查。可用[]运算符直接存取数据。 需要快速查找或使用连续且动态分配的内存空间。对两端以外的元素插入或删除不多。
list 双向链接列表，每个节点包含一个元素。列表中的每个元素均有指针指向前一个元素和下一个元素。 需要快速的插入/删除，对查找元素的速度要求不高，需要操作前端元素。
deque 在内存中不占有一块连续的空间，介于向量和列表之间，更接近向量，适用于由两端存取数据。可用[]运算符直接存取数据。 可以提供快速的元素存取。在序列中插入/删的速度除较慢。一般不需要使用双端队列，可以转而使用vector或list。
set/multiset set是一个元素集合。集合中的元素按有序的方式存储。set中没有重复的元素，但multiset中允许有重复的元素。 需要使用元素集合，而且对元素的查找、插入和删除操作都较为频繁时，就可以使用set/multiset。
map/multimap map是{键(key),值(value)}对的组成的集合。集合中的元素按键排列。multimap是允许键/值对有重复的集合。map和multimap的关系如同set和multiset之间的关系。 如果希望将键/与值相关联就可以使用map/muitimap。

迭代器变化

方法 使用导致的迭代器变化
resize 如果缩小容器则原本指向各自元素的迭代器不变，如果容器扩展，则结束迭代器失效，并且如果必须重新分配存储，则与此容器相关的所有迭代器、指针和引用也将无效
reserve 如果发生重新分配，则与容器相关的所有迭代器，指针和引用都将无效。否则，它们都将继续引用在调用之前所引用的相同元素
assign 与该容器相关的所有迭代器，指针和引用均无效
push_back 如果发生重新分配，则与容器相关的所有迭代器，指针和引用都将无效。否则，只有结束迭代器是无效的，并且保证所有迭代器、指针和对元素的引用都保持引用在调用之前所引用的相同元素
pop_back 指向被删除元素的任何迭代器，指针和引用均失效。未删除的元素关联的迭代器不变
insert 如果发生重新分配，则与容器相关的所有迭代器，指针和引用都将无效。否则，只有那些指向位置及以后的元素关联的迭代器才会失效，所有在位置之前的元素的迭代器、指针和引用都保证始终引用与调用之前所引用的元素相同的元素
erase 指定区间包含的目标删除元素及以后的元素关联的迭代器全部失效，区间之前的元素关联的迭代器、指针和引用都保证始终引用与调用前相同的元素。
swap 在两个容器中引用元素的所有迭代器，指针和引用都保持有效，并且现在引用它们在调用之前所引用的相同元素，但是在另一个容器中进行迭代。请注意，最终迭代器不引用元素，并且可能无效
clear 与该容器相关的所有迭代器，指针和引用均无效

# Array

Array容器建立在数组的理念基础之上，通过指向容器开头和结尾的迭代器(由begin和end函数返回)解决了传统数组不便于动态区分内存边界的问题，也提供了基本的元素个数获取(size)，容器是否为空(empty)，填充(fill)的判断功能。Array容器的长度是预先给定的，如果内部元素超过这个量，就会报错。下面提到的动态数组vector很好的解决了数组预先分配的空间限制元素填充的问题，其占有的空间大小是根据元素动态分配的，而且提供的功能也更多。

以下是Array容器提供的方法使用及程序运行结果:

#include <iostream>
#include <array>
using namespace std;
int main()
{
  array<int, 5> myArray = {2, 16, 77, 34, 50};
  cout << "数组的元素个数:";
  cout << myArray.size();
  cout << endl;
  cout << "数组是否为空:";
  cout << myArray.empty();
  cout << endl;
  cout << "正序遍历数组:";
  for (auto it = myArray.begin(); it != myArray.end(); ++it)
    cout << ' ' << *it;
  cout << '\n';
  cout << "倒序遍历数组:";
  for (auto it = myArray.rbegin(); it != myArray.rend(); ++it)
    cout << ' ' << *it;
  cout << '\n';
  cout << "循环遍历:" << endl;
  for (int i = 0; i < myArray.size(); i++)
  {
    cout << myArray.at(i) << ' ';
  }
  cout << endl;
  cout << "数组的第一个元素: ";
  cout << myArray.front();
  cout << endl;
  cout << "数组的最后个元素: ";
  cout << myArray.back();
  cout << endl;
  // 可以对数组中两个元素进行直接操作
  int tmp = myArray.front();
  myArray.front() = myArray.back();
  myArray.back() = tmp;
  cout << "交换元素后:" << endl;
  cout << "数组的第一个元素: ";
  cout << myArray.front();
  cout << endl;
  cout << "数组的最后个元素: ";
  cout << myArray.back();
  cout << endl;
  array<int, 5> myOtherArray;
  myOtherArray.fill(1);
  cout << "填充新数组myOtherArray:" << endl;
  for (auto &x : myOtherArray)
  {
    cout << x << ' ';
  }
  cout << endl;
  cout << "交换数组容器后打印的myArray:" << endl;
  myArray.swap(myOtherArray);
  for (auto &x : myArray)
  {
    cout << x << ' ';
  }
  cout << endl;
  return 0;
}

# Vector

与阵列(Array)相比，向量消耗更多内存以换取管理存储和以有效方式动态增长的能力。 与其他动态序列容器（deque，list和forward_lists）相比，向量在访问其元素(就像数组)方面非常有效，并且从其末尾添加或删除元素都相对高效。对于涉及在末尾以外的位置插入或删除元素的操作，它们的性能比其他方法差，并且与list和forward_lists相比，迭代器和引用的一致性更差。

以下是Vector容器提供的方法使用及程序运行结果:

#include <iostream>
#include <vector>
using namespace std;
void printArray(vector<int>);
int main()
{
  vector<int> first;
  vector<int> second;
  vector<int> third;
  first.assign(7, 100); // 7 ints with a value of 100
  vector<int>::iterator it;
  it = first.begin() + 1;
  second.assign(it, first.end() - 1); // the 5 central values of first
  int myints[] = {1776, 7, 4};
  third.assign(myints, myints + 3); // assigning from array.
  printArray(first);
  printArray(second);
  printArray(third);
  // 将向量third中的元素添加到向量first的末尾
  for (auto &x : third)
  {
    first.push_back(x);
  }
  printArray(first);
  // 删除向量中的一个元素
  third.pop_back();
  printArray(third);

  it = third.begin();
  // 在下标为1的为主插入单个数据
  third.insert(it + 1, 22);
  printArray(third);
  // 在下标为2的位置插入2个10
  third.insert(it + 2, 2, 10);
  printArray(third);
  // 在向量third的头部位置插入从向量second开始到结尾的元素
  third.insert(third.begin(), second.begin(), second.end());
  printArray(third);
  third.erase(third.begin());
  printArray(third);
  third.erase(third.begin(), third.begin() + 3);
  printArray(third);
  first.swap(third);
  printArray(first);
  printArray(third);
  first.clear();
  printArray(first);

  return 0;
}
void printArray(vector<int> array){
  if (array.size() > 0){
    for (auto &x : array){
      cout << x << ' ';
    }
    cout << endl;
  }
  else{
    cout << "向量为空" << endl;
  }
}

# List

List容器属于序列式容器，其结构基于双向链表，它的内存空间是不连续的，元素之间通过指针相连，因此在对这类容器中的元素进行插入和删除元素的效率较高。由于读取元素时需要从链表的端部遍历直到访问到目标元素，所以读取效率较低，相较于vector，list更擅长操作元素。

以下是List容器提供的方法使用及程序运行结果:

#include <iostream>
#include <list>
using namespace std;
bool notOdd(int x);
void printList(list<int>);
int main()
{
  int arr[5] = {1, 2, 3, 4, 5};
  list<int> mylist(arr, arr + 5);
  // printList(mylist);
  for (list<int>::iterator it = mylist.begin(); it != mylist.end(); it++)
    cout << *it;
  cout << endl;
  for (list<int>::reverse_iterator it = mylist.rbegin(); it != mylist.rend(); it++)
    cout << *it;
  cout << endl;
  mylist.push_back(10);
  mylist.push_front(20);
  printList(mylist);
  mylist.pop_back();
  mylist.pop_front();
  printList(mylist);
  list<int>::iterator it = mylist.begin();
  mylist.insert(++it, 10);
  mylist.insert(++it, arr, arr + 2);
  mylist.resize(20);
  printList(mylist);
  list<int> mylistx;
  for (int i = 1; i <= 3; ++i)
    mylistx.push_back(i * 10);
  //将mylistx中的元素全部剪切到mylist的当前迭代器指向的位置
  mylist.splice(it, mylistx);
  //将mylist中当前迭代器指向的元素*(++it)剪切到mylistx开头
  mylistx.splice(mylistx.begin(), mylist, ++it);
  printList(mylist);
  printList(mylistx);
  mylistx.push_back(2);    // 向mylistx插入新的元素
  cout << *(++it) << endl; //此时迭代器指向传输到的容器中新增的元素
  it = mylist.begin();
  advance(it, 6); //重定迭代器指向的元素下标
  // 选定迭代器标记的区间然后将该区间中的元素剪切到mylistx的开始位置
  mylistx.splice(mylistx.begin(), mylist, it, mylist.end());
  printList(mylistx);
  mylistx.remove(0); //按值删除元素
  mylistx.remove_if(notOdd);
  printList(mylistx);
  mylistx.push_back(2);
  //删除列表中重复的数据,可传入一个回调一个函数处理两两相关的数据
  mylistx.unique();
  printList(mylistx);
  mylist.merge(mylistx); //将mylistx剪切到mylist末尾
  mylist.sort();
  mylist.reverse();
  printList(mylist);
  return 0;
}
bool notOdd(int x){
  return x % 2;
}
void printList(list<int> List){
  if (List.size() > 0){
    for (auto &x : List){
      cout << x << ' ';
    }
    cout << endl;
  }
  else
    cout << "向量为空" << endl;
}

# Set

集合(Set)是按照特定顺序存储唯一元素的容器，其内部结构为红黑树，当元素插入树的同时将被排序。 集合中元素的值不能一次在容器中修改(元素始终为const)，但可以将其插入容器或从容器中删除。

以下是Set容器提供的方法使用及程序运行结果:

#include <iostream>
#include <set>
using namespace std;
void printSet(set<int>);
int main()
{
  int myints[] = {4, 1, 3, 2, 8};
  set<int> myset(myints, myints + 5);
  printSet(myset);
  myset.insert(5);
  printSet(myset);
  myset.erase(myset.begin());
  printSet(myset);
  set<int>::iterator itemPoint = myset.find(4);
  cout << *itemPoint << endl;
  // value_comp用于比较某个值(第一个参数)是否在第二个值(第二个参数)之前
  set<int>::value_compare myValueComp = myset.value_comp();
  set<int>::key_compare myKeyComp = myset.key_comp();
  set<int>::iterator it = myset.begin();
  // 一个较为直观的例子，其实就是比较两个参数值的大小，
  // 左侧比右侧小为真否则为假
  cout << myValueComp(200, 80) << endl;
  cout << myKeyComp(200, 80) << endl;
  do{
    std::cout << *it << ' ';
  } while (myValueComp(*(it++), *myset.rbegin()));
  cout << endl;
  for (int i = 0; i < 5; i++){
    cout << i;
    if (myset.count(i) != 0)  //在容器中搜索具有等于i的键的元素，并返回匹配数
      cout << " is an element of myset." << endl;
    else
      cout << " is not an element of myset." << endl;
  }
  // 比较比参数值小或大的值，lower_bound(4)返回4的迭代器，upper_bound(4)返回5的迭代器
  cout<<*myset.lower_bound(4)<<endl;
  cout<<*myset.upper_bound(4)<<endl;
  return 0;
}
void printSet(set<int> myset){
  for (set<int>::iterator it = myset.begin(); it != myset.end(); it++){
    cout << *it << " ";
  }
  cout << endl;
}

# Map

映射是关联容器，用于存储按特定顺序由键值和映射值的组合形成的元素。 在Map中，键值通常用于对元素进行排序和唯一标识，而映射值存储与该键关联的内容。键和映射值的类型可能有所不同。

以下是Map容器提供的方法使用及程序运行结果:

#include <iostream>
#include <map>
using namespace std;
void printMap(map<char, int>);
int main(){
  map<char, int> mymap = {{'a', 10}, {'b', 20}};
  mymap['c'] = 30;//可以按下标插入新的键值对
  printMap(mymap);
  mymap.at('a') = 100;//通过at访问map中的元素
  printMap(mymap);
  mymap.insert({'x', 90});//插入键值对
  printMap(mymap);
  auto ret = mymap.equal_range('a');//获取相等元素的范围
  cout << ret.first->first << ":" << ret.first->second << " ";
  cout << endl;
  return 0;
}
void printMap(map<char, int> myMap){
  for (map<char, int>::iterator it = myMap.begin(); it != myMap.end(); it++)
    cout << it->first << ":" << it->second << " ";
  cout << endl;
}

# Queue & Stack

堆栈和队列是容器适配器，队列支持从数据序列的一端插入，从另外一端取出，而堆栈则从数据序列的顶部添加数据并只能从顶部取出数据，容器适配器不支持任何迭代器，是初等的数据结构。

以下是Queue & Stack容器提供的方法使用及程序运行结果:

#include <iostream>
#include <queue>
#include <stack>
using namespace std;
int main(){
  queue<int> queue;
  stack<int> stack;
  for (int i = 0; i < 5; i++){
    queue.push(i);
    stack.push(i);
  }
  int sum = 0;
  // 可以通过front和back函数访问队列的两端，
  // 通过pop从队首删除元素，用push从队尾添加元素
  cout << "front: " << queue.front() << " back: " << queue.back() << endl;
  while (!queue.empty()){
    cout << queue.front() << " ";
    sum += queue.front();
    queue.pop();
  }
  cout << endl;
  cout << sum << endl;
  while (!stack.empty()){
    // 对于堆栈来说，只能通过top返回最近添加的元素，通过pop从堆栈顶部删除元素
    cout << stack.top()<<" ";
    stack.pop();
  }
  cout<<endl;
  cout << "front: " << queue.front() << " back: " << queue.back() << endl;
  return 0;
}
